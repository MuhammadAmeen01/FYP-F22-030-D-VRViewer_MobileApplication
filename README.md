# 📱 AR Interior Visualization App (Flutter)

An interactive **Flutter-based Augmented Reality (AR) mobile application** that enables users to visualize and place 3D furniture in their real-world environment using their smartphone camera.

This app helps users make better purchasing decisions by allowing them to preview how products will look and fit in their space.

---

## 🚀 Features

- 📱 Built with Flutter for cross-platform support  
- 🛋️ Place and reposition 3D objects in real space  
- 🔄 Move, rotate, and scale objects interactively  
- 👀 View objects from multiple angles  
- 📐 Improved understanding of object size and placement  
- 🎯 Real-time visualization for better decision-making  

---

## 🛠️ Technologies Used

- Flutter  
- Dart  
- ARCore / ARKit (via Flutter plugins)  
- Firebase (for fetching 3D models)  
- REST APIs (if applicable)
